## 剧情总结
“我们明日见”活动剧情紧承伦蒂尼姆战役的后续，聚焦于各方势力在战役尾声及萨卡兹大撤军期间的动向和心路历程，同时也埋下了新的伏笔。

故事起始于伦蒂尼姆城外，塔拉深池领袖爱布拉娜的妹妹拉芙希妮（苇草）带着追随者抵达，他们准备前往伦蒂尼姆。苇草决定独自行动，进入威灵顿公爵的营地，登上“加斯特里尔”号旗舰。她在途中救治了战场上的维多利亚和萨卡兹伤兵，展现了悲悯之心。在舰上，苇草面见威灵顿公爵，公爵暗示其姐爱布拉娜暂时不便相见，并询问苇草如何得知G-0高地的坐标，提及罗德岛博士。苇草表示博士是为结束战争而协助她。最终，苇草以“塔拉”的名义，向威灵顿公爵下达命令，要求其率部攻破萨卡兹防线，解救城内的维多利亚平民，彰显塔拉的荣誉。威灵顿欣然接受命令，称只要苇草愿意下令，她的号令就有效力。爱布拉娜通过威灵顿传话，邀请苇草观看一场“盛大的表演”。随后，爱布拉娜本人出现在“加斯特里尔”号舰首，释放强大的紫色火焰蔓延至伦蒂尼姆城墙，击溃萨卡兹防线，并与城墙上的食腐者之王孽茨雷进行了简短的意志交锋，展示了塔拉的力量和野心。目睹这一切的苇草，意识到战局已非自己和追随者能参与，带领队伍踏上归乡之路。

与此同时，伦蒂尼姆城内，前龙门警司陈找到了九领导的整合运动残部。这支队伍在战争中收留了大量无处可去的感染者和萨卡兹平民，包括一些俘虏的孩子。塔露拉也在队伍中。他们面临下一步的去向问题。联军士兵在城内搜捕并粗暴对待萨卡兹平民，九对此感到愤怒，试图阻止。陈则主张采取更稳妥的方式，担心队伍身份复杂会引来麻烦。九和陈就如何处理被捕萨卡兹以及队伍未来方向产生激烈争执。九认为时间紧迫，不能坐等，必须采取行动保护这些人，并寻求一个真正属于他们的地方，一个没有歧视和压迫之处，这是受Guard的影响。塔露拉介入，阻止了两人的冲突。塔露拉提出一个大胆的提议：带领这支队伍回到乌萨斯。她认为这是纠正自己过去错误的方式，尽管她的存在可能成为整合运动黑暗历史的象征，但她愿意成为新队伍的“旗帜”或“武器”，投身“真正的火焰”去偿还罪孽，在乌萨斯接受审判。九认同这个方向，认为整合运动需要一个新生。陈尽管不完全赞同其激进行为，但出于对两人的责任和对队伍变化的关注，决定作为罗德岛干员随行监视，以便向罗德岛交代。在乌萨斯雪原，有匿名电台广播感染者反抗活动的消息，提到“整合运动”的号召和一个火焰标志，这似乎与塔露拉和九的新方向产生了呼应。

在伦蒂尼姆的另一处，拥有不死之身/快速愈合体质的诺威尔正在城内寻找赋予他此能力的人（“独角兽”）。他偶然遇到一对逃难的夫妇和他们的孩子奥利，尽管声称不为救人而来，他还是带领他们前往典范军据点。路上，他们遇到了伦蒂尼姆自救军领袖克洛维希娅，她正组织市民撤离。诺威尔协助疏散。奥利走失，诺威尔和奥利母亲寻找。克洛维希娅找到奥利，但在撤离过程中遭遇废墟塌陷而牺牲，奥利被她护住得以幸存。诺威尔冒险进入废墟，找到了幸存的奥利和牺牲的克洛维希娅。在地下深处，他感应到了那个赋予他长生之力的“独角兽”的存在，并与之进行了空灵的对话。独角兽（此时显露其与克洛维希娅死亡相关的幽灵形态）表示理解诺威尔漫长生命中的痛苦，并提出补偿。诺威尔在对话中逐渐意识到独角兽就是他一直在寻找的源头，但它并非他想象中的“仙子”或“护国公”，而是一个与死亡和腐败相关的欺诈者。最终，诺威尔拒绝了独角兽，选择带着奥利回到地面。回到地面后，他发现自己因救奥利而受的伤并未像往常一样迅速愈合，意识到独角兽（克洛维希娅死后）的能力让他暂时失去了“不死”的特性。尽管得知自己错过了长生来源的本体，诺威尔最终放下了追寻死亡的执念，选择利用其仍具有的救助能力（治愈他人，而非自己）去帮助更多活着的人。

与此同时，萨卡兹血魔隐德来希抵达伦蒂尼姆，联系上凯尔希，目的是寻找失踪的亲王（特雷西斯的姐姐）。隐德来希是“玫瑰河畔”组织的成员，该组织在巴别塔倒塌后分崩离析，成员各自散去。她透露自己曾为保护其他姐妹而“假死”。隐德来希与凯尔希交换情报，凯尔希提供了亲王来过伦蒂尼姆的线索。隐德来希还协助破坏了杜卡雷在城内布置的法阵，间接帮助了罗德岛和典范军。她与另一位“玫瑰河畔”成员“小聪明”通过无线电协作，小聪明追查到亲王来过的痕迹。然而，隐德来希发现“小聪明”牺牲在废墟中，死前怀抱被毁的信号接收器。通过读取小聪明血液中的记忆，隐德来希得知亲王来过并留下了玫瑰河畔的誓约信物——她的一滴血。隐德来希推测亲王可能是来告别，或者阻止她们继续追查。她向凯尔希表达了追查亲王的决心，认为她欠姐妹一个回答，并决定重建玫瑰河畔，带领其他姐妹重回卡兹戴尔。

在圣王会西部大堂，食腐者之王孽茨雷率领最后的部队。赫德雷、维什戴尔、明椒等人与罗德岛干员（包括博士、阿米娅、凯尔希、Logos、Misery等）以及典范军和联军部队协作，准备终结孽茨雷及其散布的腐败。戴菲恩代表典范军，理解罗德岛和赫德雷的意图（杀死孽茨雷为萨卡兹撤离争取时间），为他们开路。维什戴尔占领军事委员会指挥中心，通过广播扰乱萨卡兹指挥系统，并向战场上的维多利亚军队喊话，鼓励他们进攻。孽茨雷则在圣王会西部大堂释放强大的腐败挽幛，形成难以驱散的浓雾，对抗进攻部队。罗德岛和典范军运用各自能力对抗腐败、维持通讯、救助伤员。在各方协同努力下，包括普通维多利亚工人出身的“大扳手”等士兵的英勇冲锋，蒸汽骑士的舍身撞击，以及女妖的法术，食腐者之王孽茨雷最终战败并消失。孽茨雷在临死前与感知到其存在的博士等人对话，表示战争已经结束，萨卡兹通过“阿喃那”主宰了命运，他只是享受最后的战争本身，并允许博士一行去见特雷西斯。战后，战场一片狼藉，胜利显得沉重。赫德雷、维什戴尔等人认识到萨卡兹大撤军的紧迫性，并开始利用“生命脊椎”组织行动。

故事的最后部分聚焦于萨卡兹大撤军。赫德雷等人利用“生命脊椎”带领无法长途跋涉的萨卡兹老弱病残撤离，维什戴尔则使用从伦蒂尼姆获得的神秘力量“阿喃那”制造了巨大的移动天灾云，笼罩在撤军队伍上空，作为对外界的威慑。这一突如其来的天灾云不仅笼罩了萨卡兹队伍，更同时出现在泰拉多个核心国家（哥伦比亚、莱塔尼亚、卡西米尔、拉特兰、乌萨斯）的首都上空，引发了广泛的恐慌和关注。帕尤卡卡这位探险家，对异常的天灾云感到好奇，故意靠近并被萨卡兹佣兵抓住，他提醒维什戴尔有舰队试图拦截，并被维什戴尔释放。Logos和Misery也在撤离队伍中，Misery在经过靠近拉特兰的区域时，回忆起与Outcast的往事，显得悲观。一支援助萨卡兹的舰队出现了，意图阻止拦截。千钧一发之际，阿米娅走到队伍最前列，以“魔王”身份使用黑王冠力量，升起一道不可逾越的黑墙，隔开了萨卡兹队伍和拦截舰队。她向双方宣告萨卡兹将回家，自己不是萨卡兹的统治者，但会引导他们走向终结苦难的道路，呼吁结束仇恨循环。萨卡兹士兵对这位异族魔王感到复杂情绪。拦截舰队最终退去。在这一过程中，阿米娅感知到来自拉特兰主机（或教宗）的声音/意识，“对话”发生。拉特兰教宗伊万杰利斯塔十一世感知到主机的异动，得知它正在与某个相似的存在（阿米娅的黑王冠）“对话”，并且主机预示了一个“近在眼前”的、前所未有的巨大危机。同时，有人（疑似博士）在特蕾西娅消失的虚空中，也感知到了来自拉特兰主机的预警。这为故事蒙上了一层新的阴影，预示着泰拉大陆即将面临比萨卡兹内战和维多利亚战争更严重的威胁。

整个活动通过多条叙事线，展现了伦蒂尼姆战役结束后的余波，各个人物和势力在乱局中的选择与挣扎，既有旧的终结与告别（孽茨雷的死亡、克洛维希娅的牺牲、“小聪明”的离世、诺威尔放下执念），也有新的开始与希望（塔拉的独立姿态、整合运动的新方向、玫瑰河畔的重聚、萨卡兹的大撤军、阿米娅的成长与承担），但最终以一个悬念收尾，预示着泰拉世界的未来充满未知与挑战。

*   罗德岛的Logos和Misery在萨卡兹撤军队伍中，Misery提到曾与Outcast在拉特兰附近执行任务，并打赌尝当地冰淇淋，表达了对逝去同伴的思念和遗憾。
*   帕尤卡卡是一个带着能让死魂灵讨厌的“符纸”的探险家，他对天灾充满好奇，暗示其身份和能力不一般。
*   阿米娅在升起黑墙时，萨卡兹集体感知到了她的存在和黑王冠的力量，引发了复杂的情绪反应，也暴露了“魔王”王冠被异族持有的事实。
*   拉特兰主机被称为“它”，能感知到与自身相似的存在（阿米娅/黑王冠），并与之“对话”，预示了远超当前冲突的未来危机。教宗伊万杰利斯塔十一世是能感知并理解主机预示的人。
*   特蕾西娅消失的“虚空”被再次提及，有人（疑似博士）进入其中，看到“金色的海洋”和“孤独的菱形”，并感知到了主机预警，显示此地与拉特兰主机、未来的危机有关联。

## 角色剧情概括
-   *   **苇草:** 带着塔拉追随者抵达伦蒂尼姆外围，独自进入威灵顿旗舰，以塔拉名义命令威灵顿攻城以解救维多利亚平民，展现塔拉的“高尚”和荣誉。目睹姐姐爱布拉娜展示力量。最终带领追随者离开，踏上归乡之路。
-   *   **爱布拉娜:** 深池领袖，通过威灵顿传话给妹妹苇草，并亲自在威灵顿旗舰上展示强大的紫火力量攻击伦蒂尼姆城墙，与食腐者之王短暂对峙，为塔拉争取声势。
-   *   **威灵顿公爵:** 维多利亚大公爵，已易帜投靠塔拉深池，听从爱布拉娜和苇草的命令，调动舰队准备攻城，成为塔拉展示力量和影响力的工具。
-   *   **九:** 整合运动现任领袖，在伦蒂尼姆收留战乱中的感染者和萨卡兹平民，坚持为无处可去之人寻找家园，受Guard影响，最终决定接受塔露拉的提议，带领队伍前往乌萨斯寻求新生。
-   *   **塔露拉:** 前整合运动领袖，随九的队伍行动，认同九为队伍寻找新方向的努力，提出回乌萨斯面对过去和罪孽，成为新队伍的“符号”或“武器”，以纠正错误并接受审判。
-   *   **陈:** 罗德岛干员，追查塔露拉和整合运动的踪迹至伦蒂尼姆，目睹队伍的变化和九的决心，尽管不完全赞同其方式，但出于责任决定随行监视塔露拉和九前往乌萨斯。
-   *   **诺威尔:** 拥有长生和快速愈合体质的人，在伦蒂尼姆寻找赋予他此能力的存在（独角兽），救助市民，目睹自救军领袖克洛维希娅牺牲。在地下与独角兽（克洛维希娅幽灵）对话，了解到能力的来源与死亡和腐败相关，最终放下寻求死亡的执念，选择继续利用自己的能力救助他人，尽管自身伤口不再愈合。
-   *   **克洛维希娅:** 伦蒂尼姆自救军领袖，在城内组织市民撤离，牺牲自己救下了奥利，死后其存在似乎与赋予诺威尔长生能力的“独角兽”现象产生关联。
-   *   **隐德来希:** 萨卡兹血魔，玫瑰河畔成员，为寻找失踪的亲王来到伦蒂尼姆，与凯尔希合作获取线索，破坏杜卡雷法阵，发现姐妹“小聪明”牺牲。通过血魔能力读取记忆，获知亲王来过的痕迹，推测亲王意图，决心重建玫瑰河畔并继续追查亲王。
-   *   **凯尔希:** 罗德岛高层，与隐德来希有旧交，在伦蒂尼姆向隐德来希提供亲王线索，同时利用罗德岛力量协调各方对抗萨卡兹（食腐者、杜卡雷法阵），关注亲王动向，并表达了罗德岛愿意协助隐德来希追查的善意。
-   *   **亲王（特雷西斯之姐）:** 失踪的关键人物，巴别塔倒塌后未再公开出现。通过隐德来希和“小聪明”的追查，确认她曾来过伦蒂尼姆，留下了玫瑰河畔的誓约信物，其动机和去向仍是谜团，但可能与告别特蕾西娅或阻止追查有关。
-   *   **赫德雷:** 军事委员会高级成员，曼弗雷德的副手，在伦蒂尼姆战役后组织萨卡兹老弱病残乘坐“生命脊椎”撤离，与罗德岛和典范军合作，利用“阿喃那”制造威慑，主张萨卡兹应掌握主动权，安全返回卡兹戴尔。
-   *   **维什戴尔:** 巴别塔残余成员，性格激进，与赫德雷合作组织萨卡兹撤离。占领军事委员会指挥中心并广播喊话扰乱战场，利用“阿喃那”制造跨国天灾威慑，看管曼弗雷德，对特雷西斯和军事委员会充满不满，但仍愿为萨卡兹的安全撤离出力。
-   *   **曼弗雷德:** 军事委员会高级军官，赫德雷的上级，被维什戴尔看管，认同特雷西斯利用伦蒂尼姆战役团结萨卡兹的战略，但认为自己和军事委员会须为战争罪行承担责任，并预见未来可能的复仇。
-   *   **孽茨雷:** 食腐者之王，古老的萨卡兹存在，效忠特雷西斯，在伦蒂尼姆圣王会西部大堂进行最后的抵抗，散布强大的腐败法术。将战争本身视为存在意义，最终在与罗德岛、典范军、联军等的战斗中战败，迎接了自己的终结。
-   *   **女妖:** 与孽茨雷合作，在圣王会西部大堂施展咒术，协助食腐者作战，并在孽茨雷战死后为其奏响挽歌。
-   *   **戴菲恩:** 维多利亚典范军军官，在军事委员会指挥中心外遭遇赫德雷等人，信任罗德岛，理解赫德雷为萨卡兹撤离而寻求终结孽茨雷的意图，为他们开路，表达了对战争流血的厌倦。
-   *   **阿米娅:** 罗德岛重要成员，“魔王”王冠持有者。参与圣王会西部大堂战斗。在萨卡兹大撤军遭遇拦截时，以“魔王”身份升起黑墙阻止冲突，向双方宣告萨卡兹回家并非开启新战争，而是结束仇恨，并感知到来自拉特兰主机的意识，与之“对话”。
-   *   **博士:** 罗德岛主角，参与圣王会西部大堂对抗孽茨雷的战斗，被孽茨雷感知到并允许通过。在特蕾西娅消失的虚空中感知到拉特兰主机的预警，与阿米娅一同面对新的危机预示。
-   *   **Logos:** 罗德岛干员，能力强大，协助对抗孽茨雷的腐败挽幛，通过哨音驱散浓雾、连接通讯，参与最后的撤军队伍，与Misery对话。
-   *   **Misery:** 罗德岛干员，协助对抗孽茨雷的腐败挽幛，通过刀刃破开浓雾制造通讯通路。参与萨卡兹撤军队伍，经过靠近拉特兰的区域，回忆与Outcast的往事，对未来感到悲观。
-   *   **帕尤卡卡:** 身份神秘的探险家，对天灾云产生强烈好奇，故意靠近萨卡兹撤军队伍并被捕，提醒维什戴尔危险，身上带有能影响死魂灵的物品，被维什戴尔释放，其身份和动机仍未完全揭露。
-   *   **伊万杰利斯塔十一世:** 拉特兰教宗，感知到拉特兰主机被扰动，与主机“对话”，得知主机感知到阿米娅/黑王冠的存在，并接收到主机关于即将到来的前所未有之危机的预示。
-   *   **"大扳手":** 维多利亚普通工人出身的典范军士兵，在圣王会西部大堂的战斗中展现勇气，用大扳手和获得的剑参与击杀食腐者之王，尽管受伤濒死，却在生命的最后时刻感受到了某种满足，成为普通人在战争中挣扎与奉献的缩影。
-   *   **"拖把头":** 年轻的食腐者，对战争和族群本质感到困惑，试图用“拖把布条”伪装自己以求生存。在战斗中被孽茨雷发现并送走，避免了死亡，其未来轨迹不明，代表了食腐者中非典型的一员。
