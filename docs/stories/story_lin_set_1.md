本次明日方舟活动“新潮旧浪”的剧情，围绕着龙门贫民窟权力的新旧交替展开。故事始于鼠王宣布将话事人的位子传给女儿林雨霞。这一决定如同平静湖面投入巨石，引起了贫民窟内部以及龙门近卫局的高度关注。诗怀雅局长出于维护龙门秩序的职责，也可能出于对林雨霞的某种复杂情谊，下令严密封锁贫民窟出入口，监控局势。

剧情的主舞台设在贫民窟一间即将拆除的酒吧，这里是旧势力代表“瘦子张”——张文浩的地盘。鼠王退位后，张文浩等老一辈试图利用新旧交替的机会，挑战林雨霞的权威并维护自身既得利益。张文浩通过欺凌一个负责医院项目、与林雨霞有关系的商人“汪老弟”，以此设下鸿门宴，迫使林雨霞现身。

林雨霞应邀前来，直接面对张文浩的挑衅。张文浩以林雨霞在她地盘上大兴土木（指建设医院）却未通知他为由，指责她不懂规矩、不讲情义，并暗中集结手下“胖坤”等人，试图用武力或威吓手段给林雨霞一个下马威，迫其就范或承认旧势力的话语权。他强调贫民窟有自己的规矩，并认为林雨霞与“白道”打交道、忽视老一辈是“忘本”。张文浩甚至提及与鼠王的情义，试图站在道德制高点上压制林雨霞。

然而，林雨霞早已预料到这一局面并做好了周全的准备。她冷静地回应张文浩的质疑，明确表示自己是特意没有通知他，并指出她上位是为了阻止某些人的为所欲为。随后，林雨霞开始揭露张文浩隐藏多年的罪行。她首先拿出照片，指出张文浩借龙门接收流民并协助其返乡的名义，暗中勾结国外黑帮进行大规模的人口贩卖活动。这些受害者原本是无家可归的流民，被张文浩等人贩卖到乌萨斯矿场或雷姆必拓等地，经历了十多年的苦难。林雨霞通过多年的调查，追踪到了这些罪行的线索和部分受害者。

在林雨霞揭露人口贩卖罪行的同时，她的手下（未直接出现，但通过对话得知）也出现在酒吧外围，将被张文浩叫来的胖坤及其手下控制住。更关键的是，林雨霞的手下将部分从各地找回的人口贩卖受害者带到现场，让他们指认胖坤。在诗怀雅率领的近卫局人员赶到现场处理斗殴时，林雨霞的手下当着近卫局的面，“举报”胖坤一行人的人口买卖罪行，并提供了人证和物证。这一举动使得近卫局必须依法处理，将胖坤等人和证人一并带走，锁定了张文浩的人口贩卖罪证。诗怀雅在现场听闻举报后，立刻明白了这是林雨霞精心设计的“算计”，尽管不爽，也只能按规矩办事。

林雨霞紧接着拿出第二张照片，揭示张文浩更深层的罪恶：他不仅与国外黑帮合作，还长期勾结龙门的腐败官员和商人，攫取贫民窟应得的利益。照片中显示的是张文浩、瑞翔建筑总裁李敏恒和一位王局长在宴席上交谈甚欢的场景。林雨霞指出，李敏恒的公司瑞翔建筑是贫民窟许多基建的承办方，表面上是“大善人”，实际上却与张文浩、王局长勾结，用劣质材料以次充好，骗取政府数千万投资和补助。这次的贫民窟医院项目，瑞翔建筑也试图通过非法手段中标，而主谋就是张文浩。

当张文浩接到李敏恒打来的求救电话，得知李敏恒已被近卫局带走时，诗怀雅恰好走进酒吧，将胖坤的通讯终端丢给他，证实胖坤也被捕。张文浩意识到自己大势已去，林雨霞的布局环环相扣，一举摧毁了他的犯罪网络和关系网。在林雨霞平静地报出更多可能与张文浩有染的同伙名字后，张文浩彻底放弃抵抗，闭眼接受失败。他向林雨霞表达了对鼠王的愧疚，并询问自己家人（妻子“王姨”、儿子“小张哥”）的安排，林雨霞承诺会妥善处理。最终，张文浩向诗怀雅自首，要求去见胖坤和李敏恒。

张文浩被带走后，诗怀雅与林雨霞进行了一段意味深长的对话。这段对话揭示了两人高中时期的旧识关系，以及她们都抱有让贫民窟融入龙门、让贫民窟居民生活在阳光下的共同理想。诗怀雅表示她封锁贫民窟并非完全为了抓捕林雨霞，更多是为了控制局面、保护她。她也承认林雨霞的手段有效，但强调必须接受近卫局的审查和监督，象征性地给林雨霞戴上手铐，要求她配合调查，给“今天的事”一个交代。对话中，两人互相嘲讽，但也流露出对彼此的理解和信任。
