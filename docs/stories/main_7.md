## 剧情总结
本次活动剧情“苦难摇篮”主要聚焦于切尔诺伯格核心城向龙门移动过程中，整合运动内部以及罗德岛、龙门各方势力的冲突与行动。故事开端，核心城指挥塔上，整合运动领袖塔露拉指出了萨卡兹雇佣兵W的背叛，并将其抛下核心城，W通过爆炸装置幸存。同时，感染者游击队领袖爱国者（博卓卡斯替）选择镇守核心城，等待敌人的来袭。

剧情转到龙门，龙门城主魏彦吾与凯尔希、阿米娅、陈等人讨论核心城撞击龙门的危机。核心城使用了乌萨斯的城邦识别码，意味着攻击它等同对乌萨斯宣战。魏彦吾最初试图通过外交解决，但从信使带来的消息得知乌萨斯内部政治斗争激烈，帝国议会和军方可能对核心城事件袖手旁观，甚至有意放任，将切尔诺伯格送给整合运动作为攻击龙门的工具。魏彦吾意识到龙门可能孤立无援，倾向主动出击，但凯尔希警告这可能导致龙门孤立。

在此背景下，陈与魏彦吾爆发激烈争执。魏彦吾指责贫民区对整合运动渗透的漠视和不信任，认为他们是龙门危机根源。陈则坚持贫民区居民是受害者，并揭示自己感染者的身份，表示无法再忍受魏彦吾的统治，决定脱离龙门独自前往核心城阻止塔露拉。她认为自己作为感染者和“叛徒”最适合执行这项任务。魏彦吾试图阻止她，甚至暗示将她视为龙门敌人，但陈心意已决，打破窗户从高楼跳下离开。星熊随后赶到，试图在城外出入口拦截陈，两人发生短暂冲突，星熊虽受伤但未能阻止陈的决心，最终放她离开，并承诺会照顾贫民区。

回到魏彦吾办公室，文月在中间协调，凯尔希和阿米娅与魏彦吾达成协议。罗德岛将介入切尔诺伯格核心城事件，阻止核心城撞击龙门，条件是龙门不再干涉罗德岛行动且不攻击罗德岛。罗德岛的动机在于阻止乌萨斯利用感染者挑起国家战争，避免感染者成为新的仇恨目标。凯尔希还向魏彦吾透露塔露拉是科西切公爵的继承人，并索要塔露拉父亲爱德华的遗物作为报酬。魏彦吾尽管不完全信任罗德岛，但面对严峻局势和对陈、文月的担忧，同意了合作。

罗德岛方面，阿米娅向干员们发表动员讲话，阐述介入此次危机的必要性——不仅是阻止撞击，更是为了感染者的未来，为了那些牺牲的同胞（如霜星、米莎），以及援助陈。罗德岛组织了五支小队（三作战，一侦察，一特殊作业）进行隐秘渗透。精英干员迷迭香在此活动中首次正式登场，展示了她独特的、与精神状态紧密相关的源石技艺（能感知周遭联系、具现化巨大力量），以及她因过去的创伤（失去家人、Scout等战友）而对“复仇”和“忘记”的纠结。凯尔希、阿米娅、迷迭香和博士组成核心小队，通过沙尘暴掩护、航空器佯攻等手段，成功渗透进入核心城地下结构。

在核心城内部探索过程中，罗德岛小队遇到了滞留在城的乌萨斯感染者平民以及前罗德岛干员Guard。Guard解释了爱国者游击队在切尔诺伯格废墟中努力维持秩序和保护平民的情况，但也揭示了整合运动内部其他派系（如萨卡兹雇佣兵）的残酷和对游击队的报复行为，导致平民处境艰难。凯尔希通过一场“审判”式的交涉，验证了Guard的正直，同时也刺激了那位乌萨斯感染者的情绪，促使她吐露了真相。得知霜星死亡和塔露拉可能背叛的消息后，Guard决心去警告爱国者。

为了对抗爱国者及其游击队的精锐防线，罗德岛制定了针对性的战术：利用Raidian的通讯中继能力和迷迭香的感知/歼灭能力，瘫痪游击队的传令兵体系，逼迫指挥官现身。在战术实施过程中，迷迭香因精神波动短暂失控，在凯尔希和阿米娅的帮助下平复。她们成功摧毁了大部分传令兵和伏击点，迫使爱国者现身。

爱国者（博卓卡斯替）现身并与罗德岛小队对峙。他认出了凯尔希，并透露了自己对卡兹戴尔王庭（特蕾西娅）、乌萨斯以及感染者事业的复杂情感和忠诚。他知晓塔露拉的背叛，也知道罗德岛杀死了霜星，但他选择不为个人的得失或复仇而战。他认为自己不能公开反对塔露拉，因为那会摧毁整合运动在感染者心中的合法性。他选择的道路是与罗德岛决战，通过自己的失败证明罗德岛的力量，将阻止塔露拉的任务交给他们。他坚信只有通过战争的结果才能决定对错和价值。

罗德岛与爱国者展开殊死搏斗。爱国者展现了无与伦比的强大和坚韧，即使受到重创依然战斗。在生命的最后时刻，爱国者脑海中闪回了与家人（妻子海伦、儿子格罗瓦兹尔、养女叶莲娜/霜星）的温暖回忆和痛苦诀别。阿米娅因同情而无意识地使用了她的能力，向爱国者展示了某种慰藉的幻觉。爱国者短暂平静后，拒绝了虚假的安宁，他愤怒地挣脱，并从阿米娅身上感知到了某种古老的力量和命运联系，结合自身死亡和天灾、源石等异常现象，他似乎触发了萨卡兹的“魔王”预言，并认出阿米娅是预言中的魔王。尽管如此，他最终还是停下了对阿米娅的攻击，选择相信未来，将对抗塔露拉的希望寄托于罗德岛。爱国者在战胜了内心的纠结和预言的恐怖后，轰然倒下，他的死引发了一场切尔诺伯格的大雪。

爱国者的死讯迅速传开，整合运动内部围绕塔露拉的阴谋展开了内讧。爱国者的盾卫和游击队成员，以及一些被感召的普通整合运动成员，决定继承爱国者的信念，反抗塔露拉。前罗德岛干员Guard也加入其中。罗德岛则派遣队伍协助这些反抗者维持秩序。W在被凯尔希甩飞后，也决定前往指挥塔向塔露拉复仇。梅菲斯特独自进入了安放霜星遗体的处理舱室。龙门城也因为切尔诺伯格的异常能量释放而下起了大雪。

最后，罗德岛小队兵分两路：阿米娅与迷迭香前往指挥塔寻找陈和塔露拉，凯尔希带领博士前往能源区阻止核心城运转。故事在陈即将与塔露拉重逢、魏彦吾被鼠王拦下、W卷土重来等多条线索交织中结束，预示着最终决战的到来。闪回片段揭示了科西切与魏彦吾过去关于塔露拉的对峙，以及萨卡兹摄政王与赦罪师对塔露拉和寻找“魔王”的讨论，暗示着更深层的阴谋。

## trivia
*   W的萨卡兹血统可能带有“魔族”的旧称，这称呼源于恐惧。W和萨卡兹雇佣兵乐意接受这个称呼，并希望将恐惧散播给敌人。
*   核心城有机能启动和停止的“密钥”，谢尔盖（米莎的父亲）和鲍里斯侯爵各持一把。爱国者在废城找到的是鲍里斯侯爵那把（后备方案）。
*   塔露拉的源石技艺能够中和或融化W的爆炸装置。
*   罗德岛的“综合生物处理室”用于处理干员或敌对感染者的遗体，有智能系统PRTS辅助。霜星的遗体被放在这里处理，引发了后续的雪天异象。
*   迷迭香的感知能力异于常人，能感受到人与人之间“相互交流的痕迹”、“像链子一样的线”，也能感受到情感和记忆的“颜色”。她的战斗方式是具现化巨大的、不可见的精神形体进行物理攻击。
*   陈的佩剑“赤霄”是火锻源石铸造的，能够对抗能量法术。她从高楼跳下也是她的习惯（“这不是我第一第二次走窗户了”）。
*   魏彦吾的私人武装“影卫”（黑蓑）在剧情中首次公开出现在他的办公室，可见局势之紧急。
*   罗德岛精英干员分为不同类型，如歼灭战专员（迷迭香）、侦察小队队长（Outcast, Scout, Raidian）、战术小队队长（Dobermann, Blaze）。侦察小队具有独立行动、发现/报告/处理危机的职责，且成员间有牺牲传统。
*   萨卡兹的古老仪式，如爱国者施展的或他自身体质引起的“食人”，是依托于矿石病感染和对源石异样理解的法术，蕴含死亡气味，会剥离生命活动迹象，远超现代法术范畴，且可能对迷迭香这类精神法术有反制作用（萨卡兹术师的能力）。
*   爱国者是温迪戈族萨卡兹，具有极强的身体素质、再生能力和抗源石病特性，但也会被源石病侵蚀。他的身体高结晶率可能使他自身成为一个巨大的巫术装置，发出“食人”仪式。他自称已两百多岁。
*   萨卡兹的“魔王”预言，与温迪戈、萨卡兹、死亡、命运等概念相关，似乎能被特定的源石能量爆发、仪式、或具有古老血脉的萨卡兹感知到。阿米娅在与爱国者交战中无意识触发了与预言相关的现象，并被爱国者确认为预言中的“魔王”。
*   萨卡兹的种群记忆可能通过源石结晶流传和累积，形成所谓的“预言”，能直接影响萨卡兹人的认知和行为。凯尔希对此持否定态度，认为预言是生理性残留，无参考性。
*   霜星的遗体处理过程异常，引发了龙门下雪的天象，暗示了她作为雪怪/温迪戈血统的特殊性。
*   梅菲斯特似乎对霜星的死亡处理舱室有特别关注，并试图进入其中。他因浮士德的死无法再唱歌，尝试哼唱摇篮曲。
*   科西切公爵在回忆中提到他给年幼塔露拉戴上的“铐环”，这个装置能引爆龙门城区，成为他控制魏彦吾的手段。他也提到自己会死在“正确的人手上”。
*   卡兹戴尔的摄政王和赦罪师在暗中观察塔露拉和罗德岛的行动，并寻找萨卡兹“魔王”以及准备另一位“她”（可能是指凯尔希口中特蕾西娅的继承人，或摄政王自己的棋子）。

## 角色剧情概括
-   *   **塔露拉:** 整合运动领袖，剧情开篇显示其冷酷和操纵本性，指认并试图处决W。她计划利用切尔诺伯格核心城撞击龙门，但其真实目的（毁灭整合运动）和与乌萨斯、科西切的关联逐渐暴露。爱国者和其游击队因她背叛整合运动而反抗她。
-   *   **W:** 萨卡兹雇佣兵领袖，塔露拉的副手，但早已心生反意。被塔露拉识破并丢下核心城后幸存。她对塔露拉充满厌恶和复仇心，同时也对爱国者等“理想主义者”的结局感到复杂。剧情最后，她决定返回核心城向塔露拉复仇。
-   *   **爱国者:** 整合运动游击队领袖，温迪戈萨卡兹老战士，霜星的养父。他坚守感染者的尊严和解放理想，对塔露拉的背叛心知肚明却不能公开反对。为了给罗德岛创造阻止塔露拉的机会，他选择牺牲自己与罗德岛决战。临终前，他认出阿米娅是“魔王”并触发了萨卡兹预言，最终带着对命运和塔露拉的愤怒死去。
-   *   **博士:** 游戏主角，作为罗德岛指挥官之一参与核心城渗透行动。与凯尔希、阿米娅、迷迭香等人并肩作战，是事件的见证者，对迷迭香的战斗方式、Guard的经历、爱国者的牺牲以及萨卡兹预言等有直接体验和反应。
-   *   **凯尔希:** 罗德岛重要成员，战术指挥和情报核心。与魏彦吾谈判促成罗德岛介入。她了解乌萨斯、萨卡兹和科西切的深层联系，引导罗德岛渗透行动。她理解迷迭香的创伤，也知道爱国者的复杂性。她对爱国者施加了某种温迪戈回魂术并是其遗愿的见证者，对阿米娅被认作“魔王”和预言的触发感到震惊。
-   *   **阿米娅:** 罗德岛领袖，核心城行动的主要指挥者和精神支柱。她向干员们阐述罗德岛的立场和目标。她对迷迭香、Guard等感染者同伴表现出极强的共情和关怀。与爱国者交战时，她无意识地展现了强大的力量和同情，触发了萨卡兹预言并被爱国者认作“魔王”。她深刻理解爱国者一生的痛苦和牺牲，并决心结束战争。
-   *   **迷迭香:** 罗德岛精英干员，具有特殊感知和强大法术能力，因过去的创伤而与情感和记忆疏离。她参与核心城渗透和作战，展现了残酷但受控的战斗方式。Scout的牺牲对她影响深远，她决心为失去的家人和战友复仇，但在目睹爱国者的死和随之而来的混乱后，开始反思复仇的意义，并希望理解事件背后的原因。
-   *   **陈:** 龙门近卫局警司，魏彦吾的侄女。她因贫民区事件与魏彦吾决裂，并公开自己感染者的身份，脱离龙门独自前往核心城寻找塔露拉（她的姐姐）。她与星熊发生冲突后离开，是罗德岛介入核心城的部分原因。
-   *   **魏彦吾:** 龙门城主，陈的舅舅。他面临核心城撞击龙门和乌萨斯阴谋的双重危机。他与陈的理念冲突爆发，但内心深处关心陈。他与凯尔希和阿米娅达成交易，希望罗德岛能解决危机，也暗示自己准备为龙门牺牲。他与鼠王的对峙显示了他的困境。
-   *   **星熊:** 龙门近卫局督察，陈的好友和下属。她试图阻止陈的离去，与她短暂交手受伤。她对陈的离去和感染者身份感到震惊和担忧，但理解陈的本性不会投靠整合运动，并承诺会照顾龙门。她和诗怀雅、林雨霞在龙门见证了大雪。
-   *   **Guard:** 前罗德岛干员，在切尔诺伯格事件后选择留在城内加入爱国者的游击队，保护乌萨斯感染者平民。他在仓库区遇到罗德岛小队，向他们解释了游击队和切尔诺伯格平民的处境，并得知霜星和爱国者的死讯，深受触动，决定帮助游击队反抗塔露拉。
-   *   **梅菲斯特:** 整合运动重要成员，在龙门战败后被幻影弩手救回。他进入了安放霜星遗体的处理舱室，似乎因浮士德的死而精神状态不稳定，并对罗德岛为何来此以及石棺的秘密感到好奇。
-   *   **文月:** 魏彦吾的伴侣和助手。她在魏彦吾与罗德岛的谈判中扮演关键角色，出于对陈和塔露拉的关心，主动提出由她出钱委托罗德岛，促成了合作。她对魏彦吾的内心和想法有深刻了解。
-   *   **林雨霞:** 鼠王的女儿，与星熊和诗怀雅有交集。她似乎参与了鼠王对魏彦吾的行动，并见证了龙门下雪。
-   *   **鼠王:** 龙门贫民区领袖，魏彦吾的旧识（林舸瑞）。他在关键时刻拦下魏彦吾，阻止他亲自前往核心城，显示他与魏彦吾之间复杂的合作和对抗关系。
-   *   **科西切公爵:** 塔露拉的养父和操纵者，在回忆片段中出现。他曾利用塔露拉身上的铐环要挟魏彦吾，展示了他对人性和政治的深刻理解和恶毒利用。他是塔露拉成为如今模样的重要推手。
-   *   **霜星:** 爱国者的养女，雪怪小队领袖。尽管已在龙门攻防战中牺牲，她的死亡是本次剧情的重要推动力，影响了爱国者、阿米娅、迷迭香、Guard等多人。她的遗体在罗德岛处理时引发了异常的雪天现象。
-   *   **Scout:** 罗德岛精英干员，侦察小队成员。在切尔诺伯格营救博士的任务中牺牲。他的牺牲被迷迭香和Guard提及，是迷迭香复仇动机和Guard行动选择的重要原因。
